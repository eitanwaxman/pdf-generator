# Wix App Documentation

This folder contains documentation and marketing materials for the PDF Generator Wix App.

## Files

### `wix-landing.html`
A standalone HTML landing/marketing page for the Wix app. This page includes:
- Hero section with call-to-action to install from Wix App Market
- Feature highlights (PDF generation, screenshots, desktop/mobile support, etc.)
- Use cases for different types of websites
- How it works section
- Installation CTA

**Usage:** Host this file on your website or link to it from the Wix App Market listing.

### `wix-setup.html`
A comprehensive setup guide with step-by-step instructions for:
- Installing the app from Wix App Market
- Getting a DocuSkribe API key
- Adding the widget to a Wix page
- Configuring all widget settings
- Troubleshooting common issues
- FAQ section

**Usage:** Link to this from the Wix App Market listing, support documentation, or share directly with users who need help setting up the widget.

## Integration with React App

The React app also includes these pages as components:
- `/wix` - Wix landing page (WixLandingView component)
- `/wix/docs` - Wix setup guide (WixDocsView component)

These are accessible at:
- https://www.docuskribe.com/wix
- https://www.docuskribe.com/wix/docs
