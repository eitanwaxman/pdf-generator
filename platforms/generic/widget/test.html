🚀 Docuskribe Widget Demo

Test the generic embed widget functionality

⚠️ Setup Required

  1. Log in to your Docuskribe account at http://localhost:3000 (or your deployed URL)
  2. Navigate to the Widget tab in the dashboard
  3. Create a new public API key with name "Test Widget"
  4. Add localhost:* to the authorized domains
  5. Copy the public key (starts with pk_live_)
  6. Replace YOUR_PUBLIC_KEY_HERE in the widget attributes below with your actual key
  7. Reload this page to test the widgets
Note: Make sure the backend server is running and the widget bundle is built. Run npm run build in the platforms/generic/widget directory first.

Testing Checklist

Use this checklist to verify the widget functionality:

  • ☐ Widget loads without errors (check browser console)
  • ☐ Button appears with correct text and styling
  • ☐ PDF generation works (current page)
  • ☐ Screenshot generation works
  • ☐ Custom URL PDF generation works
  • ☐ Loading state shows spinner and "Generating..." text
  • ☐ Success message appears after completion
  • ☐ File downloads automatically
  • ☐ Error handling works (try with invalid key)
  • ☐ Domain restriction works (test from unauthorized domain)
